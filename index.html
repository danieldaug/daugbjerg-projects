Projects

Here are some projects that showcase my skills and what I've accomplished as a programmer so far. Their source code is provided using Github and my Github profile is linked at the bottom of the page.

2D Videogame Demo with Godot

Basic 2D game including original animated sprites that collide, interact with each other, and take in user input. Built with the Godot Game Engine, coded with GDscript (similar to Python), and tested with Godot Unit Test extension.

3D Mobile Game "Daily Disc"

This is my current project with a colleague, which is 3D mobile game where the user throws a disc and tries to score in the nets in as few throws as possible. New island/level every day through random noise generation. The goal is to release a beta testing version by the end of December and release the game within the following few months. The final version will contain 100% original assets and coding with Godot.

Wordle with Tkinter

Game replicating the popular game, Wordle, with Python and the Python GUI, Tkinter. Built using VSCode.

Queue Buffer Model Using Semaphores

Simple C program which uses semaphores in order to send signals that allow an enqueue and dequeue action for values through a buffer limited to holding 30 values.
